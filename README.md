## 📋 วิธีการใช้งานเบื้องต้น

### 1. สำหรับลูกค้า (Customer Interface - `index.html`)
- **ส่งงาน:** กรอกข้อมูลติดต่อ รายละเอียดชุด และงบประมาณ
- **รหัส RN:** เมื่อส่งงานสำเร็จ ระบบจะออกเลข Reference Number (RN) ให้อัตโนมัติ
- **ใบเสร็จ:** ลูกค้าสามารถกด "บันทึกเป็นรูปภาพ" เพื่อเก็บหลักฐานการจ้างงานลงเครื่องได้ทันที

### 2. สำหรับเจ้าของร้าน/ช่าง (Admin Dashboard - `tailor.html`)
- **Dashboard:** ดูสรุปงานเข้าใหม่วันนี้, รายได้ประจำวัน และสถิติรวมในระบบ
- **จัดการงาน:** 
    - **งานใหม่:** กด "รับงาน" (ระบบจะให้ระบุชื่อช่างและเลือกหักสต็อกวัสดุอัตโนมัติ)
    - **กำลังเย็บ:** ติดตามงานที่กำลังดำเนินการ
    - **แจ้งงานเสร็จ:** เมื่อเสร็จสิ้น รายได้จะถูกนำไปคำนวณในระบบบัญชี
- **ระบบบัญชี:** แจกแจงค่าแรงช่าง (40%) และกำไรร้าน (60%) ให้โดยอัตโนมัติ
- **สต็อกสินค้า:** ตรวจสอบวัสดุคงเหลือ พร้อมแถบสีแจ้งเตือนเมื่อวัสดุต่ำกว่าเกณฑ์

## ⚙️ การตั้งค่าระบบ (Technical Setup)

1. **Google Sheets:**
   - สร้าง Sheet ใหม่ และนำโค้ดจาก `Code.gs` ไปวางใน Apps Script
   - ทำการ **Deploy** เป็น Web App และตั้งค่าสิทธิ์เป็น **"Anyone"**
2. **การเชื่อมต่อ:**
   - คัดลอก Web App URL ที่ได้ มาใส่ในไฟล์ `script.js` ตรงตัวแปร `SCRIPT_URL`
3. **LINE Notify:**
   - นำ Token จาก LINE Notify มาใส่ในแผ่นงาน `CONFIG` (คอลัมน์ `line_token`) เพื่อรับการแจ้งเตือนงานใหม่และการแจ้งเตือนสต็อกหมด
