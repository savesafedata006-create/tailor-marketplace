## 📋 รายละเอียดฟังก์ชันแต่ละหน้าหน้าใช้งาน

### 1. หน้าสำหรับลูกค้า (`index.html`)
หน้าจอนี้ออกแบบมาให้ลูกค้าใช้งานได้ง่ายที่สุด เหมือนการใช้งานแอปพลิเคชันทั่วไป
*   **สมัครสมาชิก/เข้าสู่ระบบ:** ลูกค้าสามารถสมัครสมาชิกด้วยเบอร์โทรศัพท์เพื่อเก็บประวัติการสั่งงานของตนเองได้
*   **การส่งงานจ้างใหม่:** ลูกค้ากรอกรายละเอียดชุดที่ต้องการตัด, เบอร์โทรติดต่อ, และงบประมาณที่ตั้งไว้
*   **ระบบใบเสร็จดิจิทัล (Digital Receipt):** เมื่อส่งงานสำเร็จ ระบบจะสร้างใบเสร็จพร้อม **รหัสงาน (RN)** ให้อัตโนมัติ ลูกค้าสามารถกดปุ่ม **"บันทึกเป็นรูปภาพ"** เพื่อเก็บเป็นหลักฐานในมือถือได้ทันที
*   **ติดตามงานของฉัน:** ลูกค้าสามารถล็อกอินเข้ามาดูได้ว่า งานที่จ้างไปอยู่ในขั้นตอนไหนแล้ว (เช่น รอดำเนินการ, กำลังเย็บ หรือ เสร็จแล้ว) และดูว่าช่างคนไหนเป็นผู้ดูแลงาน

### 2. หน้าสำหรับเจ้าของร้านและช่าง (`tailor.html`)
หน้าจอนี้เป็นหัวใจหลักในการบริหารร้าน แบ่งความสามารถตามสิทธิ์ผู้ใช้งาน:
*   **สรุปภาพรวม (Dashboard):** ดูยอดเงินรายได้วันนี้, จำนวนงานที่เข้าใหม่, และสถิติงานทั้งหมดที่ค้างอยู่ในระบบ
*   **ระบบจัดการงาน (Workflow):**
    *   **งานใหม่:** เมื่อมีลูกค้าสั่งงาน จะมีตัวเลขแจ้งเตือนสีแดง ช่างสามารถกด **"รับงาน"** พร้อมระบุชื่อตนเอง
    *   **ระบบหักสต็อกอัตโนมัติ:** ในขณะที่รับงาน ช่างสามารถเลือกวัสดุ (เช่น ผ้า หรือ กระดุม) จากคลังสินค้า ระบบจะตัดจำนวนออกให้ทันที
    *   **สถานะการทำงาน:** ช่างสามารถเปลี่ยนสถานะเป็น "กำลังเย็บ" และ "แจ้งงานเสร็จ" เมื่อทำงานเสร็จสิ้น
*   **ระบบบัญชีและค่าแรง:** ระบบจะคำนวณค่าแรงให้พนักงานอัตโนมัติ (เช่น ช่าง 40% / ร้าน 60%) เจ้าของร้านสามารถดูสรุปยอดเงินที่ต้องจ่ายให้ช่างแต่ละคนได้ทันที
*   **คลังสินค้า (Stock):** ตรวจสอบวัสดุคงเหลือ ระบบจะแสดง **แถบสีแดงแจ้งเตือน** เมื่อวัสดุชิ้นไหนใกล้หมด (ตามเกณฑ์ที่ตั้งไว้)
*   **จัดการสิทธิ์:** เจ้าของร้านสามารถปรับเปลี่ยนระดับสมาชิก (เช่น เปลี่ยนจากลูกค้าเป็นพนักงาน) ได้เอง

## 🔔 ระบบการแจ้งเตือนอัตโนมัติ (LINE Notify)

ระบบถูกเชื่อมต่อกับ LINE เพื่อให้เจ้าของร้านไม่พลาดข้อมูลสำคัญ:
1.  **เมื่อมีงานใหม่:** ระบบจะส่งข้อความแจ้งรายละเอียดงานและเบอร์โทรลูกค้าเข้ากลุ่ม LINE ทันที
2.  **เมื่อของในสต็อกใกล้หมด:** หากวัสดุชิ้นไหนถูกหักจนต่ำกว่ากำหนด ระบบจะส่งข้อความเตือนให้สั่งของเพิ่ม
3.  **เมื่อมีการล็อกอินของเจ้าของร้าน:** เพื่อความปลอดภัย ระบบจะแจ้งเตือนทุกครั้งที่มีการเข้าใช้งานส่วนบริหารร้าน
